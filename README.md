# video-edit

edit video into 10 minutes chunks

put video into `inputFile` folder, get video chunks in `outputFile` folder

## Requirement

- install yarn
- install node (v12+)
- install ffmpeg

## Testing and run

```zsh
// run js
$ yarn run edit-video

// upload to youtube
$ yarn run upload-to-youtube

// run shell script
$ sh edit-video.sh

// clear input/output file
$ yarn run clear
```
